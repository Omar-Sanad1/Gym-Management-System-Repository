# Gym-Management-System-Repository
A scalable and enterprise-level Gym Management System built with ASP.NET Core Web API, Entity Framework Core, and SQL Server.

## Author
**Omar Mahmoud Sanad**


## Technologies
- ASP .Net Core Web API
- Entity Framework Core (Code First)
- SQL Server
- Linq(Language Integrated Query)
- JWT Authentication
- AutoMapper
- BCrypt For Password Hashing
- Swagger


## Features
- Trainer Management
- Branch Management
- Membership Plans & Subscriptions
- Fitness Classes Enrollment
- Attendance Tracking
- Payment Management
- Reviews & Ratings
- JWT Authentication
- Filtering & Searching
- Pagination
- Data Seeding
- Fluent API Configurations

## Architecture
- Onion Architecture :
                      - Domain Layer (Core)
                      - Repository Layer
                      - Service Layer
                      - API Layer
  -DTOs

## How To Run
1. Clone Repository
2. Update Connection String
3. Run Migration
4. Run Project
